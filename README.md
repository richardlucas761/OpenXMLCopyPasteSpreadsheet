# OpenXMLCopyPasteSpreadsheet
A C# repo to experiment with copy-and-paste of spreadsheet tabs from a source to a destination spreadsheet.

## Original SO question

```
// Source - https://stackoverflow.com/q/50856596
// Posted by Mike F, modified by community. See post 'Timeline' for change history
// Retrieved 2025-11-24, License - CC BY-SA 4.0
```

```OpenXMLCopyPasteSpreadsheet.OpenXMLCopySheet.CopySheet```

Clones the content of a sheet into a destination file but overwrites the content of the destination file which may not be desirable.

```OpenXMLCopyPasteSpreadsheet.MergeXLSX.MergeXSLX```

Merges two spreadsheets together.

## Notes on XLSX files

The .XLSX files present were created on Office.com using the templates available and then downloaded to a local copy.
